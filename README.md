# Basic-Modal
Projeto simples para prática de criação de Modal utilizando apenas HTML e CSS.<br>
Vídeo aula, disponibilizada no Youtube no Canal Thulum.<br>
Se trata de um modal básico construido a partir de 2 titulos de sessão de nível 1 e um input type checkbox tendo um label for para abrir e outro para fechar o Modal; e também inseri um background animado pela regra @keyframes.
